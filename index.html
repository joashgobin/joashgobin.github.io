About Grug

Grug CSS is based on CUBE CSS - a CSS methodology focused on composition, utilities, blocks and exceptions. Grug CSS takes a vanilla CSS approach to CUBE with primitives centered on Layout, Spacing and Depth.

Learn: Layout

Grug has a few main layout primitives:

Stack

Used to stack items on top of each other. Uses grid layout to do so. Items can be positioned using classes like left, right, top, bottom or a combination of these. Alignment is centered horizontally and vertically by default.
